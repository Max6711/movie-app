## Description
React application for browsing movies using the TMDB API.
Built as a technical test using React and Redux-Saga.

## Features
- Popular / Airing Now / Favorites
- Search with debounce (500ms) and rate limit
- Pagination
- Movie details page
- Favorites stored in localStorage
- Keyboard-only navigation
- Mouse scrolling disabled

## Tech Stack
- React
- Redux Toolkit
- Redux-Saga
- TMDB API

## Setup
1. Clone the repository
2. Create `.env` file based on `.env.example`
3. Insert your TMDB API key
4. Run:
   ```bash
   npm install
   npm start
